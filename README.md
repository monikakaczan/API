# API

🏆 Palindrome backend API scoring service 🏆

### Tech stack: 
- [x] JavaScript
- [x] Node.js

### Testing: 
- [x] Chai

### How to run: 

```bash
git clone git@github.com:monikakaczan/API.git
cd API
npm install
npm start
go to http://localhost:3000 to view in the browser
```

### Tests:  

```bash
npm run test
```
